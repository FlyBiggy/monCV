WhatIsThatMelody

Description : Dans ce projet, il nous était demandé de créer un jeu ludopédagogique. Avec mon coéquipier Paul Degraëve, nous avons décidé de réaliser une version simplifiée de "N'oubliez pas les paroles" en utilisant des comptines pour enfants, que nous avons nommée "What is that melody". Lors de ce projet, nous avons utilisé une version de Java appelée iJava, ainsi que du Bash. Grâce à cette expérience, j'ai pu faire mes premiers pas sur GitHub pour collaborer avec mon coéquipier.
